# SpeakEasy Barber Lounge — Website Redesign

**Denver's #1 Rated Barbershop** — 5.0★ with 431+ reviews

## Stack

- Next.js 16 + App Router
- shadcn/ui + Tailwind v4
- GFS Didot (headings) + Montserrat (body)

## Pages

| Route | Purpose |
|-------|---------|
| `/` | Main landing page — booking-first UX, barber profiles, amenities, reviews |
| `/offer` | $10 off first visit — SMS code SPEAK10 |
| `/refer` | Referral program — code SHARE10, WhatsApp share |

## Design System

- **Black**: #000000 (backgrounds)
- **Gold**: #FFCC00 (accents, CTAs, headings)
- **2 fonts only**: GFS Didot + Montserrat
- **Mobile-first**: Bottom Book Now bar, sticky desktop floating CTA

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deployment

Deployed on [Vercel](https://vercel.com). Auto-deploys from `main` branch.
