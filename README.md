# SpeakEasy Barber Lounge — Website V2

**Faithful recreation + strategy-enhanced redesign** with toggle button.

## Toggle

- **Original** — Pixel-accurate recreation of the current WordPress/Elementor site using design-extract data
- **Strategy+** — New version with membership packages, Squire deep links, barber profiles, amenities, SMS capture, reviews, FAQ accordion

## Stack

- Next.js 16 + App Router + Tailwind v4
- Design-extract data from [Manavarya09/design-extract](https://github.com/Manavarya09/design-extract)
- Original site colors, fonts (GFS Didot, Montserrat), spacing from extraction

## Deploy

Pushes to `main` auto-deploy on Vercel.

## Design Data

`design-extract-original/` contains full design language extraction:
- Colors, typography, spacing, shadows, radii, breakpoints
- Component patterns, brand voice, page intent
- Screenshots, Tailwind config, CSS variables, Figma variables
